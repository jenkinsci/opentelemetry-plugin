HTTP parameters of the URL of the Kibana dashboard

The URL uses the Groovy GString Template Engine .

Available templating parameters are:

  • ${kibanaBaseUrl}
  • ${kibanaSpaceIdentifier}
  • ${kibanaDashboardTitle}
  • ${kibanaDashboardUrlParameters}
  • The OpenTelemetry resource attributes using the syntax: ${resource['resource-attribute-key']}

Default OpenTelemetry resource attributes:

  • service.name
  • service.namespace
  • service.version
  • jenkins.url

Default OpenTelemetry resource attributes can be overwritten and additional resource attributes can be defined using the "OTEL_RESOURCE_ATTRIBUTES" environment variable or the "-Dotel.resource.attributes" system property when starting the Jenkins controller.
