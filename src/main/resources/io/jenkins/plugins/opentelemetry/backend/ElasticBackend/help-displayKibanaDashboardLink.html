To install the Kibana Jenkins dashboard:

  • Download the Kibana Jenkins dashboard jenkins-kibana-dashboards.ndjson,
  • Open Kibana GUI and navigate to Stack Management / Saved Objects,
  • Click on "Import"
  • Drag & Drop the file jenkins-kibana-dashboards.ndjson in the area "Select a file to import",
  • Keep the default import settings "Check for existing objects" / "Automatically overwrite conflicts"
  • Click on the "Import" button
  • Verify on the import status flyout that the import is successful
  • Verify in the Kibana dashboard view that the "Jenkins Overview screen has been created"

To enable the Kibana Jenkins Dashboard in Jenkins:

  • check this checkbox ans save the Jenkins configuration
  • On the Jenkins homepage, verify that a new navigation menu entry with the link to the Kibana dashboard has been added
