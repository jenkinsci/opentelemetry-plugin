Additional OpenTelemetry configuration properties.

Jenkins OpenTelemetry configuration properties

Property Default value Description
otel.instrumentation.jenkins.web.enabled Boolean, default true Disable the instrumentation of Jenkins web requests (ie the instrumentation of Jenkins Stapler).
otel.instrumentation.jenkins.job.dsl.collapse.job.name Boolean, default false When using Job DSL generated jobs, make the pipeline run root span name a low cardinality name using the name "Job from seed '${job.jobDslSeedJob.fullName}'code>" rather than using "${job.fullName}"code>".
Useful when the Job DSL plugin creates a lot of jobs.
otel.instrumentation.jenkins.job.matrix.expand.job.name Boolean, default false When using Matrix Projects, the name of the combination jobs is by default collapsed to "${matrix-job-name}/execution" rather than using the full name that is generated joining the axis values of the combination

The configuration properties of the OpenTelemetry SDK are also supported. See: OpenTelemetry Docs / Java / Configure the SDK
